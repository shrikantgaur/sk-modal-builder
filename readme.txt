Lightweight modal builder with condition and event-based triggers.

== Description ==
SK Modal Builder is a lightweight plugin that allows you to create modals with conditions and event-based triggers. Easily display popups on specific pages, posts, or events like scroll, click, or time delay.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/sk-modal-builder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the modal builder from the WordPress admin menu.
